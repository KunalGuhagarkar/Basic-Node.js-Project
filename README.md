# 🚀 My Node.js Server

A clean, well-structured Node.js + Express server built
for learning backend development fundamentals.

## 🛠 Tech Stack
- Node.js
- Express.js
- dotenv

## 📦 Installation

git clone https://github.com/yourusername/my-node-server.git
cd my-node-server
npm install

## ▶️ Running the Server

# Development
npm run dev

# Production
npm start

## 🌐 API Endpoints

| Method | Endpoint     | Description        |
|--------|--------------|--------------------|
| GET    | /api/        | Welcome message    |
| GET    | /api/status  | Server health check|
